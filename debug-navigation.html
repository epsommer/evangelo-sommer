Navigation Debug Test

Open browser console and test navigation by opening:

http://localhost:3001/time-manager

In browser console, test these commands:

// Test if ViewManager context exists
console.log('ViewManager state:', window.viewManagerDebug)

// Test manual navigation
const buttons = document.querySelectorAll('button[aria-label*="Navigate"]')
console.log('Navigation buttons found:', buttons.length)
buttons.forEach((btn, i) => console.log(`Button ${i}:`, btn.textContent, btn.getAttribute('aria-label')))

// Test click events
const prevButton = document.querySelector('button[aria-label*="previous"]')
const nextButton = document.querySelector('button[aria-label*="next"]')
console.log('Previous button:', prevButton)
console.log('Next button:', nextButton)

// Test if buttons are clickable
if (prevButton) {
    console.log('Previous button click handler:', prevButton.onclick)
    console.log('Previous button event listeners:', getEventListeners(prevButton))
}

Also check:

  • Are there any console errors?
  • Do the buttons have proper click handlers?
  • Is the ViewManager context working?
  • Are the buttons visible and not covered by other elements?
